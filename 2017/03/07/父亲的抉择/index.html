父亲的抉择

发表于 2017-03-07   |     |   阅读次数

刚刚结束与父亲的每两周的固定通话，心里不禁感慨：
父亲终究是抵不过岁月的侵蚀啊。

在我的记忆里，父亲虽然个子极为矮小，但像是有永远用不完的力气一般，各种脏活累活他从来没有做不了、做不完过。
在传统的庄稼人看来，谁家的男人在水田里插秧更快，别人一天一人一亩地左右，我能插完一亩三分地，那我就理所当然的是村上能干的男人。谁家的男人在收割的时候能挑一百根、能多挑十根，谁就是村上更厉害更健壮的男人。这些通常也都意味着谁家的庄稼收成更好，这一年的收入怎样，过年能不能过的舒心。
庄稼人太辛苦。一年到头，农历三月，清明节前后就得开始准备孕育这一年第一次的收成。到六月之前，都得断断续续的负责上庄稼的肥料和杀虫除草。等到了六月，如同大学生的考试周一般，双抢开始了。抢收、抢种。收的晚了，庄稼人看着黄灿灿的、硕大的粮食往下掉，自己却收不过来，下一季的种下去也晚了。这些实在是考验庄稼人的体力，最迟一周之内得全部做完。
然而，我父亲几乎是全村最矮小的却是插秧最快的、挑水稻较多的，我们家的收成也都会是最好的，这就是我一直引以为豪的地方，父亲的那股倔劲、那股好强、不服输的劲。
但是，
父亲今年无声52了。不向人服输但不得不向时间、向老天服输，父亲，有心无力了。年轻时任性的折腾、好强，落得身体一身旧疾，每次都能“先”天气预报几步，预示到将有暴雨来袭。
上周父母两人一起去了趟广东，“困守”在落后的小县城里大半辈子的父母终于出去见识了下外边的花花世界。不知道使用智能手机、不知道用网络的便利、不会讲一口别人能轻松听懂的普通话，这些都使得这个远近出名、聪明勤俭的汉子————我的父亲无法轻易的在大城市中带着我的母亲活的够滋润。
让我感慨啊，让旁人唏嘘啊。
一个以聪明机智著称的男人、一个身材矮小却能不输给其他人的男人，去了外面寸步难行。是文化缺失导致的吗?不尽如此。还有待在小城镇里思想上的麻痹，觉得我能养家糊口、我能挣到不菲的积蓄，生活上我就足够了，也不需要、不想要、懒得去学习当代最基本的生存之道或是生存工具。父亲当下的行当里生意一年不如一年，想要转行了、想要再去外边的世界拼搏一次、至少留下自己的脚印，可是不会网络、不会手机，是硬伤啊！
再厉害的汉子封闭自己，也迟早会被时代潮流所翻涌、打翻，最终那些紧跟时代科技潮流的弄潮儿，指着抓住翻船的你，笑道：看吧，他那会也是哪般的弄潮争先。
祝福父亲。祝福父亲能够意识到，能够在这一次抓住潮流的余流，重回浪潮之巅。
